# worm
A worm 2d made in 5 days 

Menu :
Green team start

Press Left arrow or Right arrow to select ur sking
Press space to confirm or press enter to cancel

Game :
  Move
    To jump : "Z" & "Space"
    Left : "Q" & "Left arrow"
    Right : "D" & "Right arrow"
  
  Next
    Next turn : "Enter"
    Next characters : "Tab"
    
  Fire
    2 types of missiles :
      Normal : Normal missile, will fall.
      Guided : Control it with the mouse.
