# worm
A worm 2d made in 5 days 

Menu :

	Green team start

	Select skin : "Left arrow" & "Right arrow"
	Confirm : "Space"
	Cancel : "Enter"

Game :

	Move
		To jump : "Z" & "Space"
		Left : "Q" & "Left arrow"
		Right : "D" & "Right arrow"

	Next
		Next turn : "Enter"
		Next characters : "Tab"
    
	Fire
	2 types of missiles :
	  	Normal : Normal missile, will fall.
		Guided : Control it with the mouse.
